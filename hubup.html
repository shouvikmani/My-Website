HubUp






HubUp is a web application that lets users create and discover public events. HubUp makes it easy to find events, meet new people, and expand your social network. In building HubUp, I learned a lot about web development and UI design. I used technologies like Django, Google Maps API, and Bootstrap to create a functional product with a beautiful user interface. HubUp was my term project for my first CS class at CMU.
